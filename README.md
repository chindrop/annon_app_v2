# BLACKBIRD ANNOTATION TOOL
Blackbird Annotation Tool is a web application which lets its users to upload, listen, visualise and annotate audio recordings. The main aim is to provide user generated metadata on bird sound recordings for further scientific analysis. It is written in Python 3.10 using Flask  framework.
